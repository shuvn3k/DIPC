About these documents¶

These documents are generated from reStructuredText sources by Sphinx, a document processor written for the Python documentation.

Development of the documentation and its toolchain takes place on the docs@nsclient.org mailing list. We’re always looking for volunteers wanting to help with the docs, so feel free to send a mail there!

See Reporting Bugs for information how to report bugs in this documentation, or Python itself.
